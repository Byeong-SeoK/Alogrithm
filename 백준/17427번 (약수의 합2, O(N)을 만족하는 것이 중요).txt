import sys

N = int(sys.stdin.readline())

Sum = 0
for i in range(1, N+1):
    Sum = Sum + (N//i)*i
"""
N이 10이라고 가정하자
이때 1~10까지의 모든 수의 약수의 합은
1, (1,2) (1,3), (1,2,4) ~~~ (1,2,5,10)을 더한 것에 해당한다.
이때 1은 총 10개가 있고 2는 세보면 알겠지만 5개가 있고 3은 3개가 있고
이런 식으로 세다보면 10은 결국 1개가 있다.
이를 다 더하면 1*10 + 2*5 + ~~~ + 10*1 의 형태를 가지고 이 형태가 결국 약수의 합이 된다.
"""

print(Sum)
